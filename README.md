# 🦁 Telegram-бот для Московского зоопарка: «Тотемное животное»

Интерактивный Telegram-бот, разработанный для популяризации программы опеки животных «Клуб друзей» Московского зоопарка. Бот в формате увлекательной викторины помогает пользователю узнать своё «тотемное животное» среди обитателей зоопарка, рассказывает о программе опеки и предоставляет удобные инструменты для связи с сотрудниками.

## ✨ Основные возможности

- **Викторина с умным алгоритмом:** 5 вопросов с вариантами ответов. Система баллов подбирает животное, максимально похожее на пользователя по характеру.
- **Работа с медиа:** Отправка реалистичных изображений животных в соответствии с брендбуком зоопарка.
- **Программа опеки:** Подробная информация о том, как взять животное под опеку и помочь в сохранении биоразнообразия.
- **Социальные сети:** Кнопка «Поделиться результатом» для репоста в другие соцсети со встроенной ссылкой на бота.
- **Обратная связь:** Встроенный механизм сбора отзывов от пользователей для улучшения сервиса.
- **Связь с сотрудником:** Возможность задать вопрос напрямую сотруднику зоопарка (бот пересылает вопрос вместе с ID пользователя).
- **Удобный UX:** Возможность перезапустить викторину, интуитивно понятные inline-клавиатуры и дружелюбный тон общения.

## 🛠 Стек технологий

- **Язык:** Python 3.10+
- **Фреймворк:** [aiogram 3.x](https://docs.aiogram.dev/) (асинхронный бот)
- **Управление состояниями:** FSM (Finite State Machine) из aiogram
- **Конфигурация:** `python-dotenv` для безопасного хранения токенов
- **Работа с файлами:** `aiofiles` для асинхронной отправки медиа

## 🚀 Установка и запуск

### 1. Клонирование репозитория
```bash
git clone https://github.com/ВАШ_НИКНЕЙМ/zoo_quiz_bot.git
cd zoo_quiz_bot
```

### 2. Создание виртуального окружения
**Windows (PowerShell):**
```powershell
python -m venv venv
.\venv\Scripts\Activate.ps1
```
**macOS / Linux:**
```bash
python3 -m venv venv
source venv/bin/activate
```

### 3. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 4. Настройка переменных окружения
Создайте в корне проекта файл `.env` и добавьте в него токен вашего бота, полученный от [@BotFather](https://t.me/BotFather):
```env
BOT_TOKEN=123456789:ABCdefGhIJKlmNoPQRsTUVwxyZ
```

### 5. Добавление изображений (Опционально)
Для полноценной работы поместите реалистичные фотографии животных в папку `data/images/` со следующими именами:
- `manul.jpg`, `elephant.jpg`, `flamingo.jpg`
- `polar_bear.jpg`, `giraffe.jpg`, `meerkat.jpg`

*(Если изображения отсутствуют, бот корректно отправит только текстовое описание).*

### 6. Запуск бота
```bash
python main.py
```

## 📝 Настройка связи с сотрудником
Для работы функции пересылки вопросов откройте файл `routers/contact.py` и замените значение переменной `STAFF_ID` на ваш реальный Telegram ID (его можно узнать у бота [@userinfobot](https://t.me/userinfobot)).
